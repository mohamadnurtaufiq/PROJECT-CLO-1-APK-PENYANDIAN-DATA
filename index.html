Mode ECB — Electronic Codebook

Metode kriptografi yang digunakan:
1. Caesar Cipher — Geser nilai ASCII tiap karakter sebesar nilai shift.
   Formula enkripsi: C = (P + shift) mod 256
   Formula dekripsi: P = (C − shift + 256) mod 256
   Ref: Stallings, W. (2017). Cryptography and Network Security, 7th ed., hal. 34–35.
2. Transposition Cipher — Susun ulang posisi karakter tiap blok berdasarkan urutan kunci kolom.
   Ref: Stallings, W. (2017). Cryptography and Network Security, 7th ed., hal. 38–39.
Mode ECB: Tiap blok dienkripsi secara independen (tanpa chaining).
   Ref: Stallings, W. (2017). Cryptography and Network Security, 7th ed., hal. 210–212.

Mode CBC — Cipher Block Chaining

Metode kriptografi yang digunakan:
1. XOR Chaining — Setiap blok plaintext di-XOR dengan blok cipher sebelumnya sebelum dienkripsi.
   Blok pertama menggunakan IV sebagai pengganti blok cipher sebelumnya.
   Formula: Ci = Transpose(Pi XOR Ci-1), dengan C0 = IV
   Ref: Stallings, W. (2017). Cryptography and Network Security, 7th ed., hal. 214–216.
2. Transposition Cipher — Susun ulang posisi karakter dalam blok hasil XOR.
   Kunci transposisi CBC berbeda dengan ECB (syarat: min. 1 metode berbeda antar mode).
   Ref: Stallings, W. (2017). Cryptography and Network Security, 7th ed., hal. 38–39.
